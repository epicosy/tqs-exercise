# TQS Exercise

1. Clone repository:

``` console
$ git clone https://github.com/epicosy/tqs-exercise && cd tqs-exercise
```

2. Build image:

``` console
/tqs-exercise$ docker build . -t tqs-exercise
```

3. Run container:

``` console
/tqs-exercise$ docker container run -v /DIR:/tqs-exercise --it tqs-exercise
```
